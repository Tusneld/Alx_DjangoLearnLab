Introduction to Django: LibraryProject
Welcome to the LibraryProject! This project is a hands-on introduction to building web applications using the Django framework. It focuses on the core fundamentals of setting up a project, defining data structures, and managing content through the built-in admin tools.

Project Overview
This repository demonstrates the following Django capabilities:

Development Environment Setup: Initialization of a Django project (LibraryProject) and application (bookshelf).

Data Modeling: Implementation of a Book model with specific attributes (Title, Author, and Publication Year) to interact with a database.

Django ORM & CRUD: Hands-on experience using the Django Shell to perform database operations:

Creating new book records.

Retrieving and viewing data.

Updating existing entries.

Deleting records.

Admin Interface Customization: Configuring the Django Admin dashboard to provide a user-friendly interface for managing library data, including custom list views, search fields, and filters.

Project Structure
LibraryProject/: The main project configuration directory.

bookshelf/: The Django app containing the logic for the book inventory.

manage.py: The command-line utility for administrative tasks.

*.md files: Documentation of specific CRUD operations performed in the Django shell.
